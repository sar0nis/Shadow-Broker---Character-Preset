— READ ME —

INSTALLATION

Download the Archive and extract the GtS - He-Who-Whispers.json file.
Place it in: Skyrim Special Edition\Data\SKSE\Plugins\CharGen\Presets
IN-GAME USE

Launch the game and open RaceMenu.
Set race to Argonian and gender to Male.
Go to the Presets tab.
Press F9 and select the "GtS - He-Who-Whispers" preset.
Confirm.
Done.
